Cat Herder App

built with:

  • Vue.js
  • JavaScript
  • HTML
  • CSS
  • PrimeVue
  • Pinia
  • VCalendar
  • Vite
  • Supabase
  • Git
  • VS Code

I built this app as a solo-project. My idea for this project was to facilitate the task of caring for house cats - both by storing important information and dates about cats and by exchanging this information with cat sitters. It is built as a platform of information exchange between cat owners and cat sitters who are already in contact with each other, not as a platform to find cat sitters / cat owning clients.

  • user login with email and password
  • connect with other users
  • add and edit cat info, notes and dates
  • share cat data with selected users
  • responsive design
